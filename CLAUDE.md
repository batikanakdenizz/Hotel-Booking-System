# Hotel Booking System

## Communication & Language

- The user may communicate in **Turkish or English** — respond in the language they use.
- **All project artifacts must be in English**, including:
  - Source code (variable names, function names, class names)
  - Code comments and docstrings
  - Commit messages
  - Documentation (README, API docs, etc.)
  - File and folder names
  - Database schema (table names, column names)
  - UI text (unless the project explicitly requires localization)
